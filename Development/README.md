# Development

Папка для проектов разработки.

## Структура

- `emulator/` - проект эмулятора T18FL3
- `native_navigator/` - проект нативного навигатора
- `tiggo_launcher/` - проект лаунчера
- `ЯндексДог/` - проект приложения ЯндексДог

Каждый проект находится в своей ветке git. См. `ПРАВИЛА.md` для деталей.
