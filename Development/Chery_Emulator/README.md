# Chery Emulator

This project emulates the head unit (HU) of the Chery Tiggo 8 Pro Max.

- **Purpose**: Provide a local, controllable environment that reproduces behavior and constraints of the real in-car infotainment system.
- **Scope**: Screen resolution, layout, navigation integration, and interaction patterns specific to Tiggo 8 Pro Max.
- **Status**: Initial scaffold. Implementation to be added in this repository as the new codebase is designed.
