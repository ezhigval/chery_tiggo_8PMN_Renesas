# Результаты поиска приложения навигации

## Статус поиска

### Найдено:
1. **SVMapService** (`com.desaysv.mapservice`)
   - Системный сервис навигации
   - Поддерживает AutoNavi и Thinkware
   - НЕ является самим приложением навигации

2. **SVMeterInteraction** (`com.desaysv.meterinteraction`)
   - Взаимодействие с приборной панелью
   - НЕ является приложением навигации

### Не найдено:
- Само приложение навигации (TurboDog/AutoNavi)
- Package name приложения "НАВИГАЦИЯ" из лаунчера

---

## Возможные варианты

### 1. Приложение встроено в SVMapService
- Может быть Activity внутри SVMapService
- Нужно проверить все Activity в манифесте

### 2. Приложение находится в другом месте
- Может быть в пользовательских приложениях (не в extracted)
- Может быть в обновлении прошивки (soc_update.zip)
- Может быть установлено отдельно

### 3. TurboDog = AutoNavi
- TurboDog может быть просто брендингом AutoNavi
- Package name: `com.autonavi.minimap` или `com.autonavi`
- Нужно проверить, установлено ли это приложение

### 4. Приложение использует другой package name
- Может быть обфусцировано
- Может иметь неочевидное название

---
