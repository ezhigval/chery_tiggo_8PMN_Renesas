## Логика организации

### По категориям:
- **01_КОНТЕКСТ_ПРОЕКТА** - общая информация о проекте
- **02_АНАЛИЗЫ** - детальные анализы компонентов
- **03_ПЛАНЫ** - планы реализации
- **04_ИНСТРУКЦИИ** - практические руководства
- **05_СТАТУСЫ** - текущие статусы
- **06_ТЕХНИЧЕСКОЕ_ЗАДАНИЕ** - ТЗ проекта
- **07_СПРАВОЧНИКИ** - справочная информация
- **08_АРХИВ** - архивные документы
- **09_ЭМУЛЯТОР** - документация по эмулятору T18FL3
- **10_РАЗРАБОТКА** - документация разработки (ЯндексДог, Native Navigator, APK модификация)
- **11_QNX** - документация QNX окружения
- **12_T18FL3** - база знаний о системе T18FL3
- **13_QUICK_START** - быстрые старты для эмулятора и разработки
- **14_TROUBLESHOOTING** - решение проблем и отладка
- **15_СТАТУСЫ_ИСТОРИЧЕСКИЕ** - исторические статусы проектов
- **16_СИСТЕМНАЯ_ИНФОРМАЦИЯ** - системная информация T18FL3 (бэкапы, отладка, конфигурация)
- **теория/** - теоретические материалы
- **Яндекс_MapKit_NaviKit/** - документация Яндекс API

### По хронологии:
- Номера в начале названий папок (01_, 02_, и т.д.) указывают порядок изучения
- Внутри папок файлы также упорядочены по номерам

### По удобству:
- Все связанные документы в одной папке
- Четкая иерархия
- Легко найти нужный документ
- Подпапки для статусов, инструкций, анализов внутри проектов

### По проектам:
- **Эмулятор** - все про эмулятор T18FL3
- **Разработка** - все про разработку (ЯндексДог, Native Navigator, APK модификация)
- **T18FL3** - все знания о системе T18FL3
- **QNX** - все про QNX окружение
