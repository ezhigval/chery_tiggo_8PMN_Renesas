# T18FL3 Emulator

Кастомный эмулятор для головного устройства Chery Tiggo 8 PRO MAX T18FL3

## Особенности

- ✅ **Два экрана** 12.3" (Instrument Cluster + Multimedia System)
- ✅ **Полное логирование** всех процессов в реальном времени
- ✅ **CAN-шина симуляция** для имитации автомобиля
- ✅ **Работа с оригинальными образами** без изменений
- ✅ **Доступ на всех уровнях** от GUI до машинного кода

## Установка

### Требования

- Python 3.10+
- QEMU (qemu-system-aarch64)
- PyQt6

### Установка зависимостей

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Использование

1. Запустите эмулятор
2. Нажмите "Start" для запуска QEMU
3. Используйте элементы управления автомобилем:
   - Зажигание (запуск/остановка двигателя)
   - Магнитола (громкость, навигация, настройки)
   - Кнопки на руле (навигация по меню)
4. Используйте интерфейс разработчика:
   - Анализ логов в реальном времени
   - Инжекция CAN сообщений
   - Эмуляция системных событий
5. Просматривайте логи в реальном времени

## CAN-шина

Эмулятор генерирует стандартные CAN сообщения:
- Скорость автомобиля (0x100)
- Обороты двигателя (0x101)
- Температура двигателя (0x102)
- Уровень топлива (0x103)
- Климат-контроль (0x200)
- Статус дверей (0x201)
- Громкость магнитолы (0x300)
- Кнопки руля (0x301)

Все сообщения можно отправлять вручную через интерфейс разработчика.

## Логирование

Все процессы логируются в:
- Текстовый файл: `logs/t18fl3_emulator_YYYYMMDD_HHMMSS.log`
- JSON файл: `logs/t18fl3_emulator_YYYYMMDD_HHMMSS.jsonl`

## Разработка

Эмулятор спроектирован для разработки на всех уровнях:
- GUI приложения
- Android системные сервисы
- QNX процессы
- Машинный код

Все логи доступны в реальном времени для отладки.
