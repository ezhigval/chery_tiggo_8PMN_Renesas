# Ключевые классы и методы NaviKit для Android

## 1. Guidance - Основной класс навигации

### Получение данных о маршруте

- `getCurrentRoute()` → `@Nullable DrivingRoute` - текущий маршрут
- `getRouteStatus()` → `@NonNull RouteStatus` - статус маршрута
- `getAlternatives()` → `@NonNull List<Alternative>` - альтернативные маршруты

### Получение данных о местоположении

- `getLocation()` → `@Nullable Location` - текущее местоположение
- `getLocationClass()` → `@Nullable LocationClass` - класс местоположения
- `getRoadName()` → `@Nullable String` - название текущей дороги

### Получение данных о скорости

- `getSpeedLimit()` → `@Nullable LocalizedValue` - ограничение скорости на текущей дороге
- `getSpeedLimitStatus()` → `@NonNull SpeedLimitStatus` - статус ограничения скорости
- `getSpeedLimitsPolicy()` → `@NonNull SpeedLimitsPolicy` - политика ограничений для региона
- `getSpeedLimitTolerance()` → `double` - допуск превышения скорости

### Получение данных для HUD

- `getWindshield()` → `@NonNull Windshield` - данные для отображения на лобовом стекле
- `getAnnotator()` → `@NonNull Annotator` - аннотатор для карты

## 2. Windshield - Данные для HUD

### Ограничение скорости

- `getSpeedLimit()` → объект с методом `getValue()` - ограничение скорости

### Маневры

- `getUpcomingManoeuvres()` → `List<Manoeuvre>` - предстоящие маневры

## 3. LocalizedValue - Локализованное значение

### Методы

- `getValue()` → `double` - значение в единицах СИ
  - Для скорости: м/с (нужно конвертировать в км/ч: м/с × 3.6)
  - Для расстояния: метры
  - Для времени: секунды
- `getText()` → `String` - локализованный текст (например, "60 km/h")

## 4. SpeedLimitStatus - Статус ограничения скорости

### Методы (нужно проверить в документации)

- Возможно содержит методы для получения текущего ограничения скорости
- Возможно содержит информацию о превышении скорости

## 5. DrivingRoute - Маршрут

### Геометрия маршрута

- Методы для получения полилиний маршрута (нужно найти в документации)
- Методы для получения точек маршрута

### Данные о маршруте

- Методы для получения расстояния, времени, пробок

## 6. Location - Местоположение

### Координаты

- `getPosition()` → `Point` - координаты (широта, долгота)
- `getBearing()` → `float` - направление движения

## План извлечения данных

### 1. Ограничение скорости

```
Guidance.getSpeedLimit() → LocalizedValue
  → getValue() → double (м/с) → конвертировать в км/ч
  → getText() → String (например, "60 km/h")
```

### 2. Маневры

```
Guidance.getWindshield() → Windshield
  → getUpcomingManoeuvres() → List<Manoeuvre>
  → для каждого маневра: getTitle(), getSubtitle(), getDistance()
```

### 3. Маршрут (полилинии)

```
Guidance.getCurrentRoute() → DrivingRoute
  → методы для получения геометрии (нужно найти в документации)
```

### 4. Статус маршрута

```
Guidance.getRouteStatus() → RouteStatus
  → методы для получения статуса (нужно найти в документации)
```

### 5. События на карте

```
Guidance.getWindshield() → Windshield
  → методы для получения событий (камеры, дорожные события)
```

### 6. Карта

```
Guidance.getLocation() → Location
  → getPosition() → Point (координаты)
  → getBearing() → float (направление)
```
