# Подключение ГУ к Android Studio

## Шаг 1: Включите режим разработчика на ГУ

1. Откройте **Настройки** → **О системе** (или **О телефоне/О планшете**)
2. Найдите пункт **"Номер сборки"** или **"Версия ядра"**
3. Нажмите на него **7 раз подряд**
4. Появится сообщение **"Вы стали разработчиком"**

## Шаг 2: Включите USB отладку

1. Откройте **Настройки** → **Для разработчиков** (или **Система** → **Для разработчиков**)
2. Включите переключатель **"USB отладка"**
3. Если есть опции, включите также:
   - **"Отладка по USB"**
   - **"Оставаться активным при подключении USB"**

## Шаг 3: Подключите устройство

1. Подключите ГУ к компьютеру через **USB кабель**
2. На ГУ появится запрос **"Разрешить отладку по USB?"**
3. Нажмите **"Разрешить"**
4. Отметьте галочку **"Всегда разрешать с этого компьютера"** (если есть)

## Шаг 4: Проверка в Android Studio

1. Откройте **Android Studio**
2. Перейдите в: **View** → **Tool Windows** → **Device Manager**
   - Или: **Run** → **Select Device**
3. Ваше устройство должно появиться в списке устройств
4. Если устройство не видно, проверьте подключение через ADB (см. ниже)

## Альтернатива: Подключение по Wi-Fi

Если USB не работает, можно подключиться по Wi-Fi:

1. **Сначала подключите через USB один раз** (для настройки)
2. Узнайте **IP адрес ГУ**:
   - Настройки → О системе → Статус → IP адрес
   - Или: Настройки → Wi-Fi → Нажмите на текущую сеть → IP адрес
3. Выполните в терминале:
   ```powershell
   adb tcpip 5555
   adb connect <IP_АДРЕС>:5555
   ```
   Например: `adb connect 192.168.1.100:5555`

## Проверка подключения через ADB

Выполните в PowerShell:
```powershell
$env:LOCALAPPDATA\Android\Sdk\platform-tools\adb.exe devices
```

Должно появиться что-то вроде:
```
List of devices attached
ABC123XYZ    device
```

## Устранение проблем

### Устройство не видно в Android Studio

1. Проверьте, что USB отладка включена на ГУ
2. Попробуйте другой USB кабель
3. Попробуйте другой USB порт
4. Перезапустите ADB:
   ```powershell
   adb kill-server
   adb start-server
   adb devices
   ```

### Драйверы не установлены

1. Установите драйверы для вашего устройства (обычно автоматически через Windows Update)
2. Или установите **Universal ADB Driver** или драйверы от производителя ГУ

### Устройство видно как "unauthorized"

1. На ГУ должно появиться запрос на разрешение отладки
2. Нажмите **"Разрешить"** и отметьте **"Всегда разрешать"**

## После успешного подключения

1. В Android Studio выберите ваше устройство в списке устройств
2. Запустите приложение: **Run** → **Run 'app'** (или Shift+F10)
3. Приложение установится и запустится на ГУ
