# Что такое YandexNavigatorDev?

## Это НЕ полноценный навигатор!

**YandexNavigatorDev** - это **dev-версия для отладки**, созданная для:
1. **Понимания потока данных** - как работает навигация в Yandex Navigator
2. **Логирования** - отслеживания всех событий и вызовов
3. **Тестирования интеграции** - проверки работы TiggoBridgeSender
4. **Отладки** - поиска точек внедрения для модификации APK

## Где обычный интерфейс навигатора?

### Оригинальный Yandex Navigator

**Обычный интерфейс навигатора** - это **оригинальное приложение Yandex Navigator**, которое уже установлено на эмуляторе:

- **Package:** `ru.yandex.yandexnavi`
- **Приложение:** Яндекс.Навигатор (оригинальное)

Это полноценный навигатор с:
- ✅ Картами
- ✅ Построением маршрутов
- ✅ Навигацией
- ✅ Всеми функциями

### Dev-версия (YandexNavigatorDev)

**YandexNavigatorDev** (`ru.yandex.yandexnavi.dev`) - это **упрощенная версия** для отладки:
- ❌ Без карт (нужен API ключ MapKit)
- ❌ Без полноценного UI
- ✅ С логированием всех событий
- ✅ С пониманием потока данных

## Зачем нужна dev-версия?

1. **Отслеживание потока данных:**
   - Как инициализируется навигация
   - Где получаются данные
   - Как они передаются

2. **Тестирование интеграции:**
   - Проверка работы TiggoBridgeSender
   - Отправка данных в TiggoBridgeService

3. **Поиск точек внедрения:**
   - Где в оригинальном навигаторе получается NotificationDataManager
   - Куда добавить наш код для модификации APK

## Что делать дальше?

### Вариант 1: Использовать оригинальный навигатор

Для реальной навигации используйте **оригинальный Yandex Navigator**:
```bash
adb shell am start -n ru.yandex.yandexnavi/.core.NavigatorActivity
```

### Вариант 2: Добавить карту в dev-версию

Если нужно визуализировать карту в dev-версии:
1. Раскомментировать MapKit инициализацию
2. Добавить API ключ
3. Создать полноценный layout с MapView

Но это не обязательно - основная цель dev-версии - логирование и отладка.

### Вариант 3: Модифицировать оригинальный APK

Конечная цель - модифицировать **оригинальный Yandex Navigator APK**, чтобы он отправлял данные в TiggoBridgeService.

**YandexNavigatorDev** - это инструмент для отладки, а не полноценный навигатор. Для реальной навигации используйте **оригинальный Yandex Navigator** (`ru.yandex.yandexnavi`).
