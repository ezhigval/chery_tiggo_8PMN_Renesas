# Найденные пакеты системы

## ✅ TurboDog Navigation

**Package:** `com.astrob.turbodog`  
**Статус:** Найден и подтвержден  
**Дата:** 2025-11-13

### Описание
Это штатное приложение навигации TurboDog, которое мы искали. Используется системой для отображения навигации на приборной панели.

### Связанные компоненты
- **SVMapService** (`com.desaysv.mapservice`) - сервис-мост для навигации
- **SVMeterInteraction** (`com.desaysv.meterinteraction`) - взаимодействие с приборной панелью
- **Broadcast Intent:** `turbodog.navigation.system.message` - отправка навигационных данных

### Использование
Этот пакет можно использовать для:
1. Мониторинга состояния навигации
2. Перехвата данных навигации
3. Понимания механизма работы штатной системы
